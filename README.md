# Protown 🏠
## Protown web application repository 
### Key features:

 - SSR with client-side component hydration
 - No virtual DOM, PWA
 - Authentication based on JWT
 - Secure sessions, signed cookies
 - Lazy assets loading
 - Infinite scroll
 - Mobile friendly
